The signature glitch headline — clean white with offset electric/cyan ghosts. Pass a plain string child; it auto-mirrors into the ghost layers.

```jsx
<GlitchTitle size="hero" as="h1">GOYEN</GlitchTitle>
<GlitchTitle size="display">BIOGRAFÍA</GlitchTitle>
<GlitchTitle size="title" live>CONTACTO</GlitchTitle>
```

`size`: `hero | display | title` (or any CSS size). `live` adds the occasional flicker (reduced-motion safe). Titles only — never body, never the rider.
